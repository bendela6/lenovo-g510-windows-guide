★No USB stick? You have 4 other options

Pick whichever fits your situation. Method A (Reset This PC) is by far the easiest — no shopping, no burning, just a few clicks.

🔄A. Reset This PCNo media at all. Easiest.
💿B. Burn a DVDG510 has a built-in DVD drive.
📇C. SD cardG510 has an SD card slot.
🛠️D. OneKey RecoveryFactory restore (if available).
Recommendation:Try Method A first. If Windows is too broken to even open Settings, fall back to Method B (DVD) or buy a $5 USB stick and follow the USB steps below.

AReset This PC (no media needed) ⭐

If your current Windows still boots, you don't need anything external. Windows can reinstall itself.

  1. Click Start → Settings (gear icon) → Update & Security → Recovery.
  2. Under "Reset this PC", click Get started.
  3. Choose:
    • Keep my files — removes apps and settings, keeps your documents and photos.
    • Remove everything — wipes the whole drive (clean install, like the USB method).
  4. When asked how to reinstall, pick:
    • Cloud download — downloads a fresh copy from Microsoft (needs internet, ~4 GB).
    • Local reinstall — faster, uses files already on your drive.
  5. Click Reset. Wait 30–60 minutes. The laptop restarts several times by itself.
  6. When it finishes, go through the first-time setup (same as Step 6 below).
Can't open Settings?Restart the laptop. As soon as you see the Windows logo, hold the power button to force shutdown. Repeat 3 times — Windows will boot into Recovery. From there, pick Troubleshoot → Reset this PC.

BBurn Windows to a DVD 💿

The G510 has a built-in DVD drive (the slot on the right side). You need one blank disc:

  • DVD-R DL (dual-layer, 8.5 GB) — works for Windows 10 64-bit. Recommended.
  • Regular DVD-R (4.7 GB) — only fits Windows 10 32-bit.
  1. On another computer, go to microsoft.com/software-download/windows10 and run the Media Creation Tool.
  2. When asked, choose "ISO file" instead of "USB flash drive". Save it to your desktop.
  3. Put a blank DVD in your DVD burner.
  4. Right-click the saved .iso file → "Burn disc image" → pick your DVD drive → Burn.
  5. Eject. Take the DVD to the G510.
  6. Put the DVD in the G510's drive, restart, hit the Novo button, pick the DVD drive from the Boot Menu.
  7. The installer starts. Follow Step 5 below.

CUse an SD card 📇

The G510 has an SD card slot on the front. You can use an SD card (8 GB or larger) exactly like a USB stick.

  1. Plug the SD card into another computer (use an SD reader or built-in slot).
  2. Run the Media Creation Tool — it'll see the SD card as a removable drive. Pick it like you would a USB stick.
  3. Once it's ready, eject the SD card, plug it into the G510's SD slot.
  4. Use the Novo button → Boot Menu and pick the SD card.
  5. Follow Step 5 below.
Heads-up:Some older BIOSes don't boot from SD reliably. If the G510 doesn't show the SD card in the Boot Menu, switch to Method A or B.

DLenovo OneKey Recovery (factory reset) 🛠️

If your G510 still has the original Lenovo recovery partition, you can restore it to the state it was in when you bought it.

  1. Laptop OFF.
  2. Press the Novo button (pinhole on the side).
  3. Pick "System Recovery" from the menu.
  4. Follow the prompts to restore to factory state.
What you'll get:The original Windows 8 or 8.1 plus all the Lenovo bloatware that came with the laptop. Then run Windows Update to upgrade to Windows 10. Only works if the recovery partition was never deleted.

1Method E (USB): Back up your files

The steps below are the classic USB-stick method. Skip to here if you bought a USB stick or none of the methods above worked.

⚠️ ImportantReinstalling Windows erases everything on the C: drive.
  • Copy important files (photos, documents, downloads) to an external drive, another USB, or the cloud (Google Drive, OneDrive, Dropbox).
  • Don't forget: browser bookmarks, saved passwords (export them), and any program license keys.

2Make the Windows USB installer

Do this on any working computer.

  1. Plug your empty USB stick into the working computer.
  2. Open a web browser and go to: microsoft.com/software-download/windows10
  3. Click "Download tool now" (Media Creation Tool).
  4. Open the downloaded file. Click Accept.
  5. Choose "Create installation media (USB flash drive...) for another PC". Click Next.
  6. Leave the defaults (Language, Edition, 64-bit). Click Next.
  7. Choose "USB flash drive". Click Next.
  8. Pick your USB stick from the list. Click Next.
  9. Wait. It downloads Windows and copies it to the USB (15–30 minutes).
  10. When it says "Your USB flash drive is ready", click Finish.

Safely eject the USB. Done.

3Plug the USB into the Lenovo G510

  1. Shut down the G510 completely (not sleep — fully off).
  2. Plug in the power cable.
  3. Plug the USB stick into a USB port.

4Boot from the USB stick

The G510 has a little secret button called the Novo button. It's a tiny pinhole, usually on the left side of the laptop near the power cable, marked with a small curved arrow icon.

  1. Make sure the laptop is OFF.
  2. Take a paperclip or pin and gently press the Novo button for 1 second.
  3. The laptop will turn on and show this menu:
    • Normal Startup
    • BIOS Setup
    • Boot Menu
    • System Recovery
  4. Use the arrow keys to pick "Boot Menu" and press Enter.
  5. You'll see a list of drives. Pick the one that says "USB HDD" or your USB stick's name. Press Enter.
Doesn't see the USB?

Pick "BIOS Setup" instead → go to the Boot tab → change Boot Mode to UEFI (or try Legacy Support if UEFI fails) → set Boot Priority to Legacy First if needed → press F10 to save and exit → try the Novo button again.

5Install Windows

The Windows installer will start. Take your time — each screen is simple.

  1. Language / Time / Keyboard — Leave defaults. Click Next.
  2. Click "Install now".
  3. Product key screen — Click "I don't have a product key" (Windows activates automatically later if it was activated before on this laptop).
  4. Pick edition — Choose Windows 10 Home (or Pro if that's what you had). Click Next.
  5. License terms — Tick the box. Click Next.
  6. Choose "Custom: Install Windows only (advanced)". ← Important one.
  7. Pick the drive. You'll see a list of partitions:
    ⚠️ This step erases your data.Click each partition (Drive 0 Partition 1, 2, 3...) and click Delete for each — until you see just "Drive 0 Unallocated Space". Then click that and press Next.
  8. Wait. Windows copies files and restarts several times by itself (20–40 min). Don't touch anything.
If it keeps booting from USB: Pull the USB out after the first restart.

6First-time setup

After it restarts, you'll see the blue setup screens.

  1. Pick your region (country). Click Yes.
  2. Pick your keyboard layout. Click Yes, then Skip for the second one.
  3. Connect to Wi-Fi (enter your password).
  4. Choose "Set up for personal use". Click Next.
  5. Sign in with your Microsoft account — email and password.
    No Microsoft account?Click "Offline account" or "Create account" in the bottom corner.
  6. Create a PIN (4+ digits) for quick login.
  7. Privacy settings — turn things off if you want more privacy. Click Accept.
  8. Skip Cortana, OneDrive, and Office trial offers (click "No" / "Skip" on each).
  9. Wait while it finishes ("This may take a few minutes...").

You'll land on the desktop. 🎉

7Install drivers

Windows installs most drivers automatically, but G510-specific ones improve performance.

  1. Open the Edge browser. Go to: support.lenovo.com
  2. Type "G510" in the search box.
  3. Pick "Lenovo G510 Laptop (IdeaPad)".
  4. Click "Drivers & Software" → "Manual Update".
  5. Download and install (one at a time):
    • Chipset driver (Intel)
    • Graphics / VGA driver
    • Wireless LAN driver (if Wi-Fi isn't working)
    • Audio driver (if no sound)
    • Bluetooth driver
Easier option:Skip this step — let Windows Update handle drivers. Open Settings → Update & Security → Windows Update → Check for updates. Run it 2–3 times until nothing new appears.

8Update Windows

  1. Click Start → Settings (gear icon) → Update & Security.
  2. Click Check for updates.
  3. Let it download and install everything. Takes a while.
  4. Restart when asked.
  5. Repeat until it says "You're up to date".

9Install your programs

Reinstall what you use:

  • Browser: Chrome, Firefox
  • Microsoft Office (if licensed)
  • VLC — plays any video
  • 7-Zip — opens .zip and .rar files
  • Printer driver — from the printer brand's site

Copy your backed-up files back from your external drive.

?Common problems

USB stick doesn't show up in Boot Menu

Go into BIOS Setup (Novo button → BIOS Setup) → Boot tab → enable Legacy Support AND set Boot Priority to Legacy First. Save with F10. Try again.

"We couldn't create a new partition" error

Unplug all USBs except the install one. Or, on the partition step, delete ALL partitions until only one big "Unallocated Space" remains.

No Wi-Fi after install

Use a wired Ethernet cable temporarily and run Windows Update, OR download the wireless driver on another computer from Lenovo's site and copy it over with a USB.

Screen is huge / low resolution

Graphics driver isn't installed yet. Run Windows Update or install it from the Lenovo site.

Laptop is slow after install

Wait 30 minutes after first boot — Windows runs background tasks. Also: the G510 will be much faster with an SSD (~$25 upgrade) replacing the old hard drive.

📅About Windows 11

Windows 10 security updates end October 14, 2025. After that:

  • Easiest: Keep using Windows 10 — still works, just no new security patches.
  • Or: Install Windows 11 with a workaround tool called Rufus (skips the CPU check). Search "install Windows 11 on unsupported PC Rufus" for guides.
