Reinstall Windows on a Lenovo G510

Pick how you want to reinstall — only the steps you need will show up.

How do you want to reinstall?

Click a method. You can change it later.

Back up your files

⚠️ This method erases everythingThe C: drive will be wiped. Copy important files first.
Heads-upIf you choose "Keep my files", your documents are kept but all apps are removed. If "Remove everything", the drive is wiped — back up first.
  • Copy photos, documents, downloads to an external drive or the cloud (Google Drive, OneDrive, Dropbox).
  • Don't forget: browser bookmarks, saved passwords (export them), and program license keys.
  • Plug in the power cable (don't run on battery during this).

Open Reset This PC

  1. Click Start → Settings (the gear icon).
  2. Click Update & Security.
  3. On the left, click Recovery.
  4. Under "Reset this PC", click the Get started button.
Can't open Settings?Restart the laptop. As soon as you see the Windows logo, hold the power button for 5 seconds to force shutdown. Repeat 3 times — Windows will boot into Recovery mode. Then pick Troubleshoot → Reset this PC.

Pick your reset options

Windows will ask you two questions:

Question 1: What to keep?

  • Keep my files — your documents/photos stay. All apps and settings are removed.
  • Remove everything — wipes the whole drive. Like a fresh install.

Question 2: How to reinstall?

  • Cloud download — fresh copy from Microsoft (~4 GB, needs internet). Recommended if Wi-Fi is fast.
  • Local reinstall — faster, uses files already on your drive. Picks this if internet is slow.

On the confirmation screen, click Reset.

Wait for it to finish

  • Windows will restart several times by itself. Don't touch anything.
  • Takes 30–60 minutes typically.
  • Make sure the power cable stays plugged in the whole time.

When it's done, you'll see either the desktop (if you kept files) or the blue first-time setup screen.

Make the USB installer

Do this on any other working computer. You need:

  • A USB stick — at least 8 GB. Everything on it will be erased.
  • About 30 minutes.
  1. Plug the USB into the working computer.
  2. Open a browser, go to microsoft.com/software-download/windows10
  3. Click "Download tool now" (Media Creation Tool).
  4. Open the downloaded file. Click Accept.
  5. Choose "Create installation media for another PC" → Next.
  6. Leave defaults (Language, 64-bit) → Next.
  7. Choose "USB flash drive" → Next.
  8. Pick your USB stick from the list → Next.
  9. Wait (15–30 min). Click Finish when done. Eject safely.

Burn the DVD

Do this on any other computer with a DVD burner. You need:

  • A blank DVD-R DL (dual-layer, 8.5 GB) — fits Windows 10 64-bit.
  • Or a regular DVD-R (4.7 GB) — only fits 32-bit.
  1. Go to microsoft.com/software-download/windows10 and run the Media Creation Tool.
  2. Choose "ISO file" instead of "USB flash drive". Save it to Desktop.
  3. Put the blank DVD in the burner.
  4. Right-click the saved .iso file → "Burn disc image" → pick the DVD drive → Burn.
  5. Eject when done. Take it to the G510.

Make the SD card installer

Do this on any other working computer. You need:

  • An SD card — 8 GB or larger. Will be erased.
  • An SD reader (or built-in slot on the other computer).
  1. Plug the SD card into the working computer.
  2. Go to microsoft.com/software-download/windows10, run the Media Creation Tool.
  3. Choose "Create installation media for another PC" → Next.
  4. Choose "USB flash drive" (the SD card will appear in the list — Windows treats it the same).
  5. Pick your SD card → Next. Wait.
  6. Eject when done.
Heads-up:Some BIOSes don't boot reliably from SD. If the G510 doesn't list it in the Boot Menu, switch to USB or DVD.

Boot the G510 from your media

  1. Shut down the G510 completely (not sleep).
  2. Plug in the power cable.
  3. Insert your media (USB stick into a USB portDVD into the DVD drive on the right sideSD card into the SD slot on the front).
  4. Find the Novo button — a tiny pinhole on the left side near the power cable, with a small curved arrow icon.
  5. With the laptop OFF, use a paperclip to press the Novo button for 1 second.
  6. A menu appears. Use arrow keys to pick "Boot Menu" → Enter.
  7. Pick your media from the list ("USB HDD" or your stick's nameyour DVD driveyour SD card) → Enter.
Doesn't see your media?

Press Novo → pick "BIOS Setup" → go to the Boot tab → enable Legacy Support AND set Boot Priority to Legacy First. Press F10 to save and exit. Try Novo button again.

Install Windows

  1. Language / Time / Keyboard — leave defaults. Next.
  2. Click "Install now".
  3. Click "I don't have a product key" (Windows activates automatically later if it was activated before).
  4. Pick edition: Windows 10 Home (or Pro) → Next.
  5. Tick the license box → Next.
  6. Choose "Custom: Install Windows only (advanced)".
  7. Partitions screen:
    ⚠️ This step erases your data.Click each partition (Drive 0 Partition 1, 2, 3...) and click Delete for each — until only "Drive 0 Unallocated Space" remains. Click that, then Next.
  8. Wait (20–40 min). The laptop restarts several times. Don't touch anything.
If it keeps booting from your media:Pull out the media after the first restart.

Run OneKey Recovery

  1. Make sure the laptop is OFF.
  2. Plug in the power cable.
  3. Find the Novo button — pinhole on the left side near the power cable.
  4. Press the Novo button with a paperclip for 1 second.
  5. From the menu, pick "System Recovery" → Enter.
  6. Follow the prompts to restore to factory state. Wait (30–60 min).
What you'll get:The original Windows 8 or 8.1 with all the original Lenovo bloatware. You'll likely want to run Windows Update to get to Windows 10 afterward.
Doesn't work?If "System Recovery" isn't in the Novo menu, or fails, the recovery partition was deleted at some point. You'll need to use one of the other methods.

First-time setup

You'll see the blue setup screens. Go through them:

  1. Pick your region → Yes.
  2. Pick your keyboard layout → Yes, then Skip for the second one.
  3. Connect to Wi-Fi (enter password).
  4. Choose "Set up for personal use" → Next.
  5. Sign in with your Microsoft account.
    No Microsoft account?Click "Offline account" or "Create account" in the bottom corner.
  6. Create a PIN (4+ digits) for quick login.
  7. Privacy settings — turn off what you don't want → Accept.
  8. Skip Cortana, OneDrive, and Office trial offers.
  9. Wait for "This may take a few minutes...".

You'll land on the desktop. 🎉

Install drivers

Windows installs most drivers automatically, but G510-specific ones improve performance.

Easy path:Skip this step — let Windows Update handle it (next step). Only come back here if Wi-Fi or sound isn't working.
  1. Open Edge. Go to support.lenovo.com
  2. Search "G510".
  3. Pick "Lenovo G510 Laptop (IdeaPad)".
  4. Click "Drivers & Software" → "Manual Update".
  5. Download and install (one at a time):
    • Chipset driver (Intel)
    • Graphics / VGA driver
    • Wireless LAN driver (if no Wi-Fi)
    • Audio driver (if no sound)
    • Bluetooth driver

Update Windows

  1. Start → Settings → Update & Security.
  2. Click Check for updates.
  3. Let it download and install everything. Takes a while.
  4. Restart when asked.
  5. Repeat until it says "You're up to date".

Reinstall your programs

Things you probably want:

  • Browser: Chrome, Firefox
  • Microsoft Office (if licensed)
  • VLC — plays any video
  • 7-Zip — opens .zip and .rar files
  • Printer driver — from your printer brand's website

Copy your backed-up files back from your external drive.

✓You're done!

Your Lenovo G510 is reinstalled. If you kept your files, your documents are still there — just reinstall any apps you need.

If you want a quick reference for problems, keyboard shortcuts, or info about Windows 11, see below.

★Troubleshooting & reference

USB / DVD / SD doesn't show up in Boot Menu

Novo → BIOS Setup → Boot tab → enable Legacy Support AND set Boot Priority to Legacy First. Save with F10.

"We couldn't create a new partition" error

Unplug all USBs except the installer. Or, on the partition step, delete ALL partitions until only one "Unallocated Space" remains.

No Wi-Fi after install

Use a wired Ethernet cable temporarily and run Windows Update, OR download the wireless driver on another computer from Lenovo's site and copy it over.

Screen is huge / low resolution

Graphics driver isn't installed yet. Run Windows Update or install it from the Lenovo site.

Laptop is slow after install

Wait 30 minutes — Windows runs background tasks. Also: the G510 will be much faster with an SSD upgrade (~$25).

What about Windows 11?

Windows 10 support ends October 14, 2025. After that: keep using Win 10 (still works), or install Win 11 using Rufus (skips CPU check).
